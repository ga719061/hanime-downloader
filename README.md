# Hanime 影片下載器

一個功能完整且 UI 精美的 Chrome 擴充插件，可以下載 hanime1.me 網站的影片。

## ✨ 特色功能

- 🎯 **自動偵測影片** - 在影片頁面自動識別影片資訊
- 🎬 **多解析度選擇** - 支援多種畫質選項（720p, 1080p 等）
- 🎨 **現代化 UI** - 深色主題 + 紫粉漸層 + 玻璃擬態效果
- ⚡ **流暢動畫** - 所有互動都有精心設計的動畫效果
- 📥 **一鍵下載** - 選擇解析度後即可快速下載

## 🚀 安裝方式

### 步驟 1: 載入擴充插件

1. 開啟 Chrome 瀏覽器
2. 在網址列輸入 `chrome://extensions/`
3. 開啟右上角的「開發人員模式」
4. 點擊「載入未封裝項目」
5. 選擇 `hanime-downloader` 資料夾

### 步驟 2: 使用擴充插件

1. 前往 [hanime1.me](https://hanime1.me)
2. 開啟任一影片頁面（網址格式：`https://hanime1.me/watch?v=xxxxx`）
3. 點擊瀏覽器工具列上的擴充插件圖示
4. 在彈出視窗中選擇想要的解析度
5. 點擊「開始下載」按鈕
6. 選擇儲存位置即可開始下載

## 🎨 UI 設計特色

### 視覺風格
- **配色方案**: 深色背景 (#0f0f1e) + 紫粉漸層 (#667eea → #764ba2)
- **玻璃擬態**: 半透明卡片效果配合背景模糊
- **柔和陰影**: 多層次陰影提升立體感
- **圓角設計**: 統一 12px 圓角營造現代感

### 互動效果
- **Hover 動畫**: 卡片上移 + 陰影加深
- **選中狀態**: 藍色邊框光暈效果
- **載入動畫**: 流暢的旋轉 spinner
- **成功動畫**: 縮放彈跳效果

## 🔧 技術細節

- **Manifest V3**: 使用最新的 Chrome 擴充插件規範
- **Vanilla JS**: 無需任何外部依賴，純原生 JavaScript
- **現代 CSS**: CSS Variables + Flexbox + Grid
- **響應式設計**: 適配不同尺寸視窗

## ⚠️ 注意事項

1. **網站限制**: 此擴充插件僅在 hanime1.me 網站上運作
2. **頁面要求**: 需要在影片頁面（`/watch?v=xxxxx`）才能使用
3. **瀏覽器權限**: 首次使用時會要求下載權限
4. **檔案命名**: 下載的檔案會自動命名為「影片標題_解析度.mp4」

## 🐛 疑難排解

### 擴充插件無法載入影片資訊
- 確保您在 `hanime1.me/watch?v=xxxxx` 格式的頁面上
- 重新整理頁面後再試一次
- 檢查瀏覽器控制台是否有錯誤訊息

### 下載失敗
- 確認網路連線正常
- 檢查是否有足夠的磁碟空間
- 嘗試選擇其他解析度

### 擴充插件圖示未顯示
- 確認 `icons` 資料夾中有所有圖示文件
- 重新載入擴充插件
