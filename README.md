# Hanime 影片下載器

一個功能完整且 UI 精美的 Chrome 擴充插件，可以下載 hanime1.me 網站的影片。

## ✨ 特色功能

- 🎯 **自動偵測影片** - 在影片頁面自動識別影片資訊
- 🎬 **多解析度選擇** - 支援多種畫質選項（720p, 1080p 等）
- 📊 **即時下載進度** - 顯示下載百分比、速度和剩餘時間
- ⚙️ **個人化設定** - 預設畫質、自訂檔名規則、通知開關
- 🚀 **快速下載按鈕** - 在影片頁面直接點擊下載，無需開啟 popup
- 🎨 **現代化 UI** - 深色主題 + 紫粉漸層 + 玻璃擬態效果
- ⚡ **流暢動畫** - 所有互動都有精心設計的動畫效果
- ❌ **取消下載** - 隨時可以取消進行中的下載

## 🚀 安裝方式

### 步驟 1: 載入擴充插件

1. 開啟 Chrome 瀏覽器
2. 在網址列輸入 `chrome://extensions/`
3. 開啟右上角的「開發人員模式」
4. 點擊「載入未封裝項目」
5. 選擇 `hanime-downloader` 資料夾

### 步驟 2: 使用擴充插件

**方法一：快速下載按鈕（推薦）**
1. 前往 [hanime1.me](https://hanime1.me)
2. 開啟任一影片頁面
3. 點擊頁面上的「🚀 快速下載」按鈕
4. 選擇想要的解析度即可開始下載

**方法二：透過 Popup**
1. 點擊瀏覽器工具列上的擴充插件圖示
2. 在彈出視窗中選擇想要的解析度
3. 點擊「開始下載」按鈕

### 步驟 3: 個人化設定

1. 右鍵點擊擴充插件圖示 → 選項
2. 或在 popup 右上角點擊齒輪按鈕
3. 可設定：
   - 預設畫質（自動/1080p/720p/480p/360p）
   - 檔案命名規則（支援 `{title}`、`{id}`、`{quality}` 變數）
   - 下載完成通知開關

## 🎨 UI 設計特色

### 視覺風格
- **配色方案**: 深色背景 (#0f0f1e) + 紫粉漸層 (#667eea → #764ba2)
- **玻璃擬態**: 半透明卡片效果配合背景模糊
- **柔和陰影**: 多層次陰影提升立體感
- **圓角設計**: 統一 12px 圓角營造現代感

### 互動效果
- **Hover 動畫**: 卡片上移 + 陰影加深
- **選中狀態**: 藍色邊框光暈效果
- **載入動畫**: 流暢的旋轉 spinner
- **進度條動畫**: Shimmer 光澤效果
- **成功動畫**: 縮放彈跳效果

## 🔧 技術細節

- **Manifest V3**: 使用最新的 Chrome 擴充插件規範
- **Vanilla JS**: 無需任何外部依賴，純原生 JavaScript
- **模組化設計**: 獨立的 downloadManager 處理下載邏輯
- **現代 CSS**: CSS Variables + Flexbox + Grid
- **響應式設計**: 適配不同尺寸視窗

## ⚠️ 注意事項

1. **網站限制**: 此擴充插件僅在 hanime1.me 網站上運作
2. **頁面要求**: 需要在影片頁面（`/watch?v=xxxxx`）才能使用
3. **瀏覽器權限**: 首次使用時會要求下載和通知權限
4. **檔案命名**: 下載的檔案會根據設定自動命名

## 🐛 疑難排解

### 擴充插件無法載入影片資訊
- 確保您在 `hanime1.me/watch?v=xxxxx` 格式的頁面上
- 重新整理頁面後再試一次
- 檢查瀏覽器控制台是否有錯誤訊息

### 快速下載按鈕沒有出現
- 等待頁面完全載入
- 重新整理頁面
- 確認擴充插件已啟用

### 下載失敗
- 確認網路連線正常
- 檢查是否有足夠的磁碟空間
- 嘗試選擇其他解析度

### 擴充插件圖示未顯示
- 確認 `icons` 資料夾中有所有圖示文件
- 重新載入擴充插件
